The Digital Islands part of Cridhe Crois-Shlighe is about developing and supporting digital skills on North Uist, Benbecula and South Uist. The vision for the project is broad: teach code to children, teach code to adults, support existing projects and develop software and websites for local endevours. Follow @Digital_Islands


Adult Classes: Coding classes for adults running in Daliburgh and Lionacleit. Aimed at beginners, the first block starts early September and will focus on Python. Course booking will be through LCC. Initial enquiries to: neil@taigh-chearsabhagh.org

Coder Dojo: Code Club for young people (7 - 18) at Claddach Kirkibost Centre. All abilities welcome. Runs the third Tuesday in every month. 5.30 - 7pm *Free*

Schools: Program of after school code clubs and coding classes. Content focuses on using Scratch to create games, animations and music and introductions to basic Python and Javascript coding.
