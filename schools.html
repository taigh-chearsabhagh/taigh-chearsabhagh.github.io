We've been running after school code clubs at Sgoil an Iochdair, Sgoil Dhalabroig, Sgoil Uibhist a Tuath and delivering classes in coding at Sgoil Bhaile a' Mhanaich and Sgoil Dhalabroig. These activities will continue (and expand) throughout the coming school year. Coding classes have focussed on learning Scratch to create games, animations and music and introductions to basic Python and Javascript coding.
